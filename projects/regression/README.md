Food Delivery Time Prediction

Overview

This project predicts food delivery time using machine learning models. Two models were implemented and compared to evaluate their performance.

Models Used

- Linear Regression
- Logistic Regression

Tech Stack

Python, Pandas, NumPy, Scikit-learn, Matplotlib

Workflow

Data preprocessing → Model training → Model comparison → Result analysis

Goal

Understand how different machine learning models perform when predicting delivery time.
